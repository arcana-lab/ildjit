void IRVM_newBackendMethod ( IRVM_t *  _this,
void *  method,
t_jit_function *  backendFunction,
JITBOOLEAN  isRecompilable,
JITBOOLEAN  isCallableExternally 
)

Create a new empty backend-dependent function with signature specified by backendFunction .

Notice that this means that the signature has to be created before calling this function.

Signatures can be created by calling IRVM_translateIRMethodSignatureToBackendSignature like in the following case:

1 IRVM_translateIRMethodSignatureToBackendSignature(this, IRSignature, backendFunction);
2 IRVM_newBackendMethod(this, method, backendFunction, JITTRUE, JITTRUE);
Parameters
_thisIR virtual machine
methodPointer to set as metadata to backendFunction
backendFunctionMemory to store the new backend-dependent function
isRecompilableJITTRUE or JITFALSE
isCallableExternallyJITTRUE if the created method can be called from outside the generated machine code, JITFALSE otherwise.
