Partial compilation scheme
Overview » Compilation schemes
Collaboration diagram for Partial compilation scheme:

You can generate the IR code just for a part of the program. This section describes how.


Clean the IR code cache

First we need to set the IR code cache directory. To see how, read this IR code cache .

Now we clean the code cache by erasing it entirely. This operation is going to remove all IR code cache stored inside the IR code cache, even the ones about other programs.

$ ildjit --clean-code-cache


Choose what to compile in IR

To choose the part of the program to generate the IR for, we tag the methods to compile. For this purpose, we run the program to identify this set of methods.

$ ildjit -P1 myProgram.cil

This command has executed myProgram.cil and recorded the set of methods executed in the code cache.


Generate the IR

We now generates the IR just for the methods previously tagged:

$ ildjit -P1 -O3 --aot -x -s -m --optimizationleveltool=default myProgram.cil

This command has generated, optimized and stored the IR to the code cache.


Execute the IR code previously generated

Now we can execute the IR previously generated as many times as we want. To do so, we execute the machine code generated starting from the IR previously stored:

$ ildjit --aot -x myProgram.cil

These last two passes (i.e., IR code generation and code execution) can be collapsed by executing the following command:

$ ildjit -P1 -O1 --aot -x myProgram.cil
