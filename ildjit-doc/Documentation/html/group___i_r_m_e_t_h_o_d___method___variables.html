XanHashTable* IRMETHOD_getAllVariableDefinitions ( ir_method_t *  method)

Get all definitions of all variables.

Return all definitions of all variables used in method method.

The caller of this function is in charge to free the returned table by using the following code:

1 XanHashTableItem *item;
2 XanHashTable *allDefs;
3 XanList *defsOfVar5;
4 
5 // Fetch all definitions of all variable
6 allDefs = IRMETHOD_getAllVariableDefinitions(method);
7 
8 // Fetch all definitions of variable 5
9 defsOfVar5 = xanHashTable_lookup(allDefs, (void *)(JITNUINT)(5 + 1));
10 if (defsOfVar5 != NULL){
11  XanListItem *i;
12  i = xanList_first(defsOfVar5);
13  while (i != NULL){
14  ir_instruction_t *def;
15  def = i->data;
16  printf("Instruction %u defines variable 5\n", IRMETHOD_getInstructionPosition(def));
17  i = i->next;
18  }
19 } else {
20  printf("There is no definitions of variable 5\n");
21 }
22 
23 // Free the memory
24 item = xanHashTable_first(allDefs);
25 while (item != NULL){
26  XanList *defs;
27  defs = item->element;
28  xanList_destroyList(defs);
29  item = xanHashTable_next(allDefs, item);
30 }
31 xanHashTable_destroyTable(allDefs);

JITUINT32 IRMETHOD_getMethodLocalsNumber ( ir_method_t *  method)

Get the number of local variables.

Return the number of variables of the method defined within the bytecode program (i.e., local variables). Notice that local variables are defined by the bytecode program (e.g. CIL), which means they are the variables specified by the user within the source program (e.g. C, C#, etc...) Moreover, local variables do not include temporany variables.
