ir_value_t IRSYMBOL_resolveSymbol ( ir_symbol_t *  symbol)

Translate a symbol to its actual value.

Resolve the symbol given as input (i.e., symbol).

Resolving a symbol is done by generating its actual value in the current running instance of the ILDJIT system.

void* IRSYMBOL_unresolveSymbol ( ir_symbol_t *  symbol)

Translate a symbol to its registered value.

Translate a symbol to its registered value (i.e., the one given as input at the function IRSYMBOL_createSymbol ).

For example, the following code will properly execute without aborting for the included assertion:

1 // Create my data
2 MyData_t *myData = ...
3 
4 // Register a symbol
5 ir_symbol_t *sym = IRSYMBOL_createSymbol(myTag, myData);
6 
7 // Retrieve the original value
8 MyData_t *myData2 = IRSYMBOL_unresolveSymbol(sym);
9 assert(myData2 == myData);

void* IRSYMBOL_unresolveSymbolFromIRItem ( ir_item_t *  item)

Translate a symbol to its registered value.

If item includes a symbol, then it translates this symbol to its registered value (i.e., the one given as input at the function IRSYMBOL_createSymbol ). In this case, this function returns the same value as the one returned by IRSYMBOL_unresolveSymbol .

Otherwise, if item does not include a symbol, then it returns NULL.
