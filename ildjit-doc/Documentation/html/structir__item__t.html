Detailed Description

An IR item is a parameter to an instruction that can be a variable, constant or symbol.

Field Documentation

JITUINT16 internal_type

Internal type of the parameter. This field always identifies the type of the parameter and cannot be equal to IROFFSET.

JITINT16 type

Type of the parameter. If the parameter is a variable, then this field is equal to IROFFSET.

TypeDescriptor* type_infos

If the parameter is a value type, this field points to its CIL type description.

ir_value_t value

Value of the IR constant, ID of the IR variable or ID of the IR symbol.
