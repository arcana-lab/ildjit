JITBOOLEAN IRMETHOD_canInstructionBeReachedFromPosition ( ir_method_t *  method,
ir_instruction_t *  inst,
ir_instruction_t *  position 
)

Check whether a given instruction can be reacheed from a given position of the CFG.

Return JITTRUE if instruction inst can be reached from position; JITFALSE otherwise.

If inst can be reached from position, then it means that after the execution of position, inst can be executed.

On the other hand, if inst cannot be reached from position, then it means that after executing position, inst cannot be executed.

This function requires REACHING_INSTRUCTIONS_ANALYZER .

Parameters
methodIR method
instInstruction that is checked if it can be reached.
positionPosition in the CFG
Returns
JITTRUE or JITFALSE

Return the list of call instructions of a method.

Return the list of call instructions that compose the method method .

An instruction is a call one if the following function returns JITTRUE:

1 IRMETHOD_isACallInstruction

The caller is in charge of freeing the returned list by calling the xanList_destroyList function of the XanList data structure.

1 XanList *l;
2 
3 // Fetch the set of call instructions of a method
4 l = IRMETHOD_getCallInstructions(method);
5 
6 // Use the list
7 ...
8 
9 // Free the memory
10 xanList_destroyList(l);

Every element of the list is a pointer to the data structure ir_instruction_t .

The memory pointed by these addresses contains the up-to-date information of the instructions (i.e., changing the instructions of the returned list will change the instructions of the method as well).

XanHashTable* IRMETHOD_getInstructionsThatCanBeReachedFromPosition ( ir_method_t *  method,
ir_instruction_t *  position 
)

Get the set of instructions that can be reached from a given position of the CFG.

Return the set of instructions that can be reached from a given position of the CFG.

Each instruction in the table returned, say inst, can be reached from position; hence, after the execution of position, inst can be executed.

The caller is in charge of freeing the returned table by calling xanHashTable_destroyTable .

An example of code is the following:

1 XanHashTable *t;
2 
3 // Fetch the set of instructions that can be reached from position
4 t = IRMETHOD_getInstructionsThatCanBeReachedFromPosition(m, position);
5 
6 // Check if a given instruction can be reached from position
7 if (xanHashTable_lookup(t, inst) != NULL){
8  // The instruction inst can be reached from position
9 
10 } else {
11  // The instruction inst cannot be reached from position
12 }
13 
14 // Free the memory
15 xanHashTable_destroyTable(t);

This function requires REACHING_INSTRUCTIONS_ANALYZER .

Parameters
methodIR method that includes position and all instructions returned.
positionPosition in the CFG
Returns
Set of instructions reachable from position
