JITNINT PLATFORM_getPlatformInfo ( struct utsname *  infos)

get some info about current platform.


NOTE: In windows is not possible to retrieve some informations.
All infos are stored in utsname structure. Inside utsname you can find current fields:

  • char[] sysname: Name of this implementation of the operating system.

char[] nodename: Name of this node within the communications network to which this node is attached, if any.

  • char[] release: Current release level of this implementation.
  • char[] version: Current version level of this release.
  • char[] machine: Name of the hardware type on which the system is running see uname() linux function for more infos.
