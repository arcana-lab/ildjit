JITUINT32  elementID

Add an element to a GEN set.

Add elementID to the set setID stored in the data flow sets.

For example, if we are using data flow with sets per instruction and these sets include variables (e.g., to compute variable defined), then:

1 // Fetch an instruction
2 ir_instruction_t *i = IRMETHOD_getInstructionAtPosition(method, 5);
3 
4 // Check if the instruction defines a variable
5 if (IRMETHOD_doesInstructionDefineAVariable(method, i)){
6 
7  //Fetch the variable defined by the instruction
8  varID = IRMETHOD_getVariableDefinedByInstruction(method, i);
9 
10  // Add the variable varID to the GEN set of the instruction i
11  DATAFLOW_addElementToGENSet(sets, IRMETHOD_getInstructionPosition(i), varID);
12 }
Parameters
setsData flow
setIDSpecific set of sets to consider.
elementIDElement to add to the setID set of sets.
