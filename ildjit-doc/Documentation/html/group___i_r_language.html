Detailed Description

Understand and manipulate the IR language.

The IR language is composed by data types and instructions. Values (always typed) can be constants or stored within IR variables.

In Instruction set you can find information about IR instructions.
In Data types you can find information about IR data types (variables and constants).
The IR code can be managed by using the IR API described in Code API .

Eventually IR code is translated to machine code of the underlying platform by relying on the IR backend described in IR backend .

To use this API, the following header must be included.

#include <ir_method.h>
