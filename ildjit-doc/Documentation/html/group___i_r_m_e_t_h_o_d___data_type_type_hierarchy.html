JITBOOLEAN IRDATA_isAssignable ( TypeDescriptor *  typeID,
TypeDescriptor *  desiredTypeID 
)

Check if a variable of a given type can be assigned to another variable with another given type.

This function returns JITTRUE if a variable of type typeID can be assigned to another variable of type desiredTypeID .

Return JITFALSE otherwise.

Parameters
typeIDType of a source variable
desiredTypeIDType of a destination variable
Returns
JITTRUE or JITFALSE
