Detailed Description

Information about pre- and post dominance between instructions can be found by using the API described in this section.

Function Documentation

ir_instruction_t* IRMETHOD_getImmediatePredominator ( ir_method_t *  method,
ir_instruction_t *  inst 
)

Returns the immediate predominator.

Returns the immediate predominator of the instruction given as input. The caller should ensure that the predominator tree has been computed before the invocation of this function. In order to ensure it, you should call the PRE_DOMINATOR_COMPUTER plugin and you should ensure that after that, no one has modified the Control Flow Graph of the method.

Parameters
methodMethod that includes inst
instInstruction to consider
Returns
Immediate predominator of inst
