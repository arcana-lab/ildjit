Detailed Description

An ir value holds either a float or non-float value.

Field Documentation

IR_ITEM_FVALUE f

Floating point value of the parameter. This field has to be considered only if the parameter is a constant of type IRFLOAT32, IRFLOAT64 or IRNFLOAT.

IR_ITEM_VALUE v

Value of the parameter. If the parameter is a variable, then the value is the identification of the variable
