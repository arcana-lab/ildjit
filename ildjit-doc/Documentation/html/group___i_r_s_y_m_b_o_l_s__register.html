Register functions for a symbol tag.

Register the set of functions given as input as the ones to use to manage all symbols of type tag.

This function returns JITTRUE if the registration succeeded; JITFALSE otherwise.

A registration can fail if previously someone has been already registered to resolve all symbols of type tag.

Parameters
tagTag of the symbol to register functions for
resolveHelper function that resolves the symbol to an actual value
serializeHelper function that converts the symbol into a format that can be stored
dumpHelper function that writes the symbol in a human-readable format
deserializeHelper function that restores the symbol from a stored format
Returns
JITTRUE or JITFALSE
