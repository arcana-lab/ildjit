Make a new node.Make a new node of the N-ary tree.

Parameters
allocFunctionThis function is called each time the tree needs to allocate new memory; if it is NULL, then the tree will use the malloc function.
freeFunctionThis function is called each time the tree needs to free memory; if it is NULL, then the tree will use the free function.
cloneFunctionThis function is called each time the tree needs to clone the variable stored inside its elements; if it is NULL, then the tree does not clone the variable stored inside each element of the tree rooted at the current node.
Returns
Return the new node.
