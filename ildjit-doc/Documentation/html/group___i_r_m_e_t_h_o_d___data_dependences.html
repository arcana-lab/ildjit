XanList* IRMETHOD_getDataDependencesFromInstruction ( ir_method_t *  method,
ir_instruction_t *  fromInst 
)

Get instructions that an instruction depends from.

Returns the set of dependences of the instruction fromInst.

If no dependences exist, it returns NULL.

Each element in the list is (data_dep_arc_list_t *). These elements are pointers to internal data structures. Therefore, modifying them will effect the system.

The set of instructions included in these elements are the instructions that need to be executed before fromInst.

The caller is responsible to destroy the returned list as following:

1 XanList *deps;
2 
3 // Fetch the dependences
4 deps = IRMETHOD_getDataDependencesFromInstruction(method, fromInst);
5 
6 // Use the dependences
7 // ...
8 
9 // Free the memory
10 xanList_destroyList(deps);
Parameters
methodMethod that includes fromInst
fromInstInstruction to consider
Returns
Dependences of frominst
