Detailed Description

The instructions that belong to this set perform calls to methods.

Macro Definition Documentation

#define IRCALL   27

Call the IR method specified by the first parameter.

The first parameter is the method ID of the called method, which can be obtained as following:

1 ir_method_t *method;
2 ...
3 IR_ITEM_VALUE methodID;
4 methodID = IRMETHOD_getMethodID(method);

#define IRLIBRARYCALL   28

Call the native method provided by the virtual machine.
The body of the method specified by the first parameter is not implemented in IR; instead is stored inside the virtual machine.

#define IRNATIVECALL   31

Call the native method specified by the third parameter (the second parameter is the name of this method and it is helpfull only for debugging purpose).
The difference with IRLIBRARYCALL is that the body of the method can be any native method, not only from methods specified by some CIL library.
