void IRMETHOD_addInstructionCallParameter ( ir_method_t *  method,
ir_instruction_t *  inst,
ir_item_t *  newCallParameter 
)

Add a new call parameter to an instruction.

Add a new call parameter cloning newCallParameter .

The new call parameter is added to the instruction inst .

For example, the following code adds a new call parameter to an existing call instruction:

1 ir_item_t newParam;
2 memset(&newParam, 0, sizeof(ir_item_t));
3 newParam.value.v = 5;
4 newParam.type = IRINT32;
5 newParam.internal_type = newParam.type;
6 IRMETHOD_addInstructionCallParameter(method, callInstruction, &newParam);
Parameters
methodIR method that includes the instruction inst
instIR instruction that includes the nth call paramter
newCallParameterThe new call parameter to clone

ir_instruction_t *  inst,
JITUINT32  nth,
ir_item_t *  from 
)

Copy an IR item to a call parameter of an instruction.

Copy an already existing parameter from into the nth call parameter of the IR instruction inst.

Call parameters start from 0 .

The nth call parameter must exist before this function is invoked.

There is no constrain about the IR item from : it could be part of any IR instruction.

methodIR method that includes the instruction inst
instIR instruction that includes the nth call paramter
nthNumber of the call parameter to consider

Set a parameter or result of an instruction.

Set the call parameter parameterNumber of the instruction inst , which belongs to the IR method method .

The first call parameter is 0 .

The typeInfos can be NULL if no information is known.

The call parameter must already exist.
