JITUINT32 IRMETHOD_getInstructionPosition ( ir_instruction_t *  inst)

Get the instruction position.

Return the position of the method where the instruction inst has been stored. Pay attention on the fact that if some instruction has been inserted, deleted, moved, then the position of each other instruction will be changed potentially. For this reason, if you need a real identificator of an instruction, then please use its memory address.

Parameters
instIR instruction to consider
Returns
position within the IR method of inst where it is stored
XanHashTable* IRMETHOD_getInstructionPositions ( ir_method_t *  method)

Get the positions of all instructions of a given IR method.

Return the positions of all instructions of method at the time the call is performed.

The returned table is the mapping from the instruction pointer to the position number plus 1.

The returned table can be used to cache the positions of instructions before a given transformation. For example:

1 XanHashTable *t;
2 t = IRMETHOD_getInstructionPositions(method);
3 // Code analysis
4 ...
5 
6 // Code transformation that can affect instruction positions
7 ...
8 
9 // Code that need instructions position before the transformation
10 JITUINT32 previousPosition;
11 previousPosition = ((JITUINT32)(JITNUINT)xanHashTable_lookup(t, inst)) - 1;

The caller is in charge of freeing the returned table by calling the following function:

1 xanHashTable_destroyTable(t);

ir_instruction_t** IRMETHOD_getInstructionsWithPositions ( ir_method_t *  method)

Return the set of instructions that compose a method.

Return the set of <instruction, position> pairs that compose the method method .

The index of the returned array is the position of the instruction pointed by the pointer stored in that array element.

The caller is in charge of freeing the returned list by calling freeFunction .

1 ir_instruction_t **insns;
2 
3 // Fetch all instructions
4 insns = IRMETHOD_getInstructions(method);
5 
6 // Print instructions
7 for (JITUINT32 i=0; i < IRMETHOD_getInstructionsNumber(method); i++){
8  ir_instruction_t *insn;
9 
10  // Fetch the instruction stored at position i
11  insn = insns[i];
12 
13  printf("Instruction stored at position %u: %s\n", i, IRMETHOD_getInstructionTypeName(insn->type));
14 }
15 
16 // Free the memory
17 freeFunction(insns);

Every element of the returned array is a pointer to the data structure ir_instruction_t . The memory pointed by these addresses contains the up-to-date information of the instructions (i.e., changing the instructions of the returned list will change the instructions of the method as well).

methodIR method that contains the returned list of instructions
Returns
Instructions that compose method
