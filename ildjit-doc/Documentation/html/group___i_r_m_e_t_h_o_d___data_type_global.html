XanList* IRDATA_getGlobals ( void  )

Fetch the list of all globals.

This function returns all globals registered at the time the call is made.

The list must be freed by the caller by invoking xanList_destroyList .

Each element of the list is a pointer to ir_global_t . These pointers actually point to the data structures that describe globals and that they will be used to generate machine code.
