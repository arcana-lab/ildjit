Create a new global.

Create a new global for the program that is compiling.

Store a global to a parameter of an instruction.

Store the global global to the specified parameter of the instruction inst.

Parameter zero represents the return parameter.

The global must be the output of the call IRGLOBAL_createGlobal

For example,

1 ir_global_t *g;
2 ir_item_t *irItem;
3 
4 // Create the global to store
5 g = IRGLOBAL_createGlobal("MyGlobal", IRUINT32, IRDATA_getSizeOfIRType(IRUINT32), JITFALSE, NULL);
6 
7 // Store the symbol
8 IRGLOBAL_storeGlobalToInstructionParameter(m, inst, 1, g, NULL);
Parameters
methodMethod that contains the parameter inst.
instInstruction where the global will be stored to.
toParamParameter of inst where the global will be stored to.
globalGlobal to store
typeInfosOptional high level descriptor of the global global.
