void IRMETHOD_destroyMemoryAliasInformation ( ir_method_t *  method)

Destroy memory alias information.

This function frees the memory used to store the memory alias information.

It does not free the memory used to store data dependences. In order to destroy data dependences as well, call IRMETHOD_destroyDataDependences .

Check if two variables may alias.

This function checks if variable i and variable j can be alias at the position in the CFG identified by the instruction inst.

Parameters

instPosition of the CFG where the alias relation is checked
iFirst variable of the alias relation
jSecond variable of the alias relation
Returns
JITTRUE if variable i may store the same address of the variable j at runtime; JITFALSE otherwise
