Allocate a new variable.

Allocate a new variable and stores its information in newVariable.

For example, the following code allocates a new integer 32 bits IR variable to use to store the result of an operation:

1 ir_item_t *resultInst;
2 resultInst = IRMETHOD_getInstructionResult(inst);
3 IRMETHOD_newVariable(method, resultInst, IRINT32, NULL);

The same effect can be achieved by calling the function IRMETHOD_setInstructionParameterWithANewVariable as following:

1 IRMETHOD_setInstructionParameterWithANewVariable(method, inst, IRINT32, NULL, 0);

IR_ITEM_VALUE IRMETHOD_newVariableID ( ir_method_t *  method)

Get a new variable ID.

Return a free variable ID to use.

This function can be used to create new IR variables to use.

For example, the following code allocates a new integer 32 bits IR variable to use to store the result of an operation:

1 ir_item_t *resultInst;
2 resultInst = IRMETHOD_getInstructionResult(inst);
3 (resultInst->value).v = IRMETHOD_newVariableID(method);
4 resultInst->type = IROFFSET;
5 resultInst->internal_type = IRINT32;

The same effect can be achieved by calling the function IRMETHOD_setInstructionParameterWithANewVariable as following:

1 IRMETHOD_setInstructionParameterWithANewVariable(method, inst, IRINT32, NULL, 0);

Set how many variables are used by a IR method.

Set the information about the highest ID used for variables inside the method. This number corresponds (if there is not segmentation over IDs) to the number of variables needed by the IR method given as input.
