Reset the fields of a parameter of an instruction.

Erase to the initial values each fields of the parameter parameter of the instruction inst .

Parameter zero represents the return parameter.

Swap two parameters of an IR instruction.

Swap two parameters of an IR instruction.

Parameter zero represents the return parameter.

For example, consider the following code:

1 IRMETHOD_swapInstructionParameters(method, inst, 1, 2);

The above code transforms the instruction

1 IRMOVE(var1, var2)

to

1 IRMOVE(var2, var1)
