Substitute a label of an IR instruction.

Substitute every reference of label old_label of the instruction given as input with the label new_label .

Both old_label and new_label are IR label IDs (i.e., IRLABELITEM ).

void IRMETHOD_substituteLabels ( ir_method_t *  method,
ir_instruction_t *  inst,
IR_ITEM_VALUE  new_label 
)

Substitute every labels of an IR instruction.

Substitute every reference of any label of the instruction given as input with the label new_label .

The parameter new_label has to be an IR label ID (i.e., IRLABELITEM ).

The difference with IRMETHOD_substituteLabel is that this function substitutes any label rather than a specific one.

For example, consider the following IR instruction:

1 branch_if_pc_not_in_range(1, 2, 3)

1, 2 and 3 are labels (i.e., IRLABELITEM ).
By calling

1 IRMETHOD_substituteLabels(method, inst, 5)

the instruction becomes

1 branch_if_pc_not_in_range(5, 5, 5)

On the other hand, by calling

1 IRMETHOD_substituteLabel(method, inst, 1, 5)

the instruction becomes

1 branch_if_pc_not_in_range(5, 2, 3)

void IRMETHOD_substituteUsesOfVariableInsideInstruction ( ir_method_t *  method,
ir_instruction_t *  inst,
IR_ITEM_VALUE  fromVarID,
IR_ITEM_VALUE  toVarID

Substitute a variable within its uses within an IR instruction.

Substitute every reference of variable fromVarID of the instruction given as input with the variable toVarID that correspond to a use of fromVarID .

The difference with IRMETHOD_substituteVariableInsideInstruction is that this function does not change the variable within the result of inst .

For example, consider the following IR instruction:

1 var 5 = add (var 5, var 2)

By calling

1 IRMETHOD_substituteUsesOfVariableInsideInstruction(method, inst, 5, 10)

the instruction becomes

1 var 5 = add (var 10, var 2)

On the other hand, by calling

1 IRMETHOD_substituteVariableInsideInstruction(method, inst, 5, 10)

the instruction becomes

1 var 10 = add (var 10, var 2)
