void IRMETHOD_shiftVariableIDs ( ir_method_t *  method,
JITINT32  offset_value 
)

Shift the variable IDs.

IR instructions use variables which are identified by a progressive number called ID of the variable (e.g. var 5). This function shifts by offset_value every variable ID of the IR method method given as input. This function performs the same task as the function IRMETHOD_shiftVariableIDsHigherThanThreshold excepts that it targets every variable rather than a subset.

Parameters
methodIR method to consider
offset_valueDelta to add to every variable IDs

Redirect branches from a label to another one.

Substitute every reference of label old_label inside every branch instruction with the label new_label .

Both old_label and new_label are IR label IDs (i.e., IRLABELITEM ).
