void IRMETHOD_setBranchDestination ( ir_method_t *  method,
ir_instruction_t *  inst,
IR_ITEM_VALUE  labelID 
)

Set the branch destination of a jump instruction.

This function set the branch destination of a IRBRANCH , IRBRANCHIF, IRBRANCHIFNOT and IRBRANCHIFPCNOTINRANGE instruction to the label given as input.

Parameters
methodIR method that includes the instruction inst
instIR instruction to consider
labelIDlabel to use
