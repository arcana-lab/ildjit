ir_item_t* IRMETHOD_getIRItemOfVariableDefinedByInstruction ( ir_method_t *  method,
ir_instruction_t *  inst 
)

Get the IR item that includes the variable defined by an instruction.

If the instruction inst may define a variable (any one), then this function returns the IR item that includes the variable ID that this instruction defines. Otherwise, this method returns NULL.

The returned IR Item is a pointer to the internal structure of the instruction, therefore:

  • the caller cannot frees it and
  • modifying the returned data structure corresponds to modify the instruction inst
