Detailed Description

In order to allocate a stack, we have to write the following:

XanStack *myStack;
myStack = xanStack_new(malloc, free, NULL);



In order to push element onto a stack, we have to write:

xanStack_push(myStack, element);



In order to view the top of the stack, without altering the stack, we have to write:

myType *myData;
myData = (myType *) xanStack_top(myStack);



In order to deallocate a stack, without destroying the items on it, we have to write:

xanStack_destroyStack(myStack);
