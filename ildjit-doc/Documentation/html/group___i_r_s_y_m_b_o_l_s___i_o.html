JITUINT32  tag,
void *  mem,
JITUINT32  memBytes 
)

Cache the serialization of a symbol.

Cache the serialization of symbol sym.

All next invocations of IRSYMBOL_deserializeSymbol will return mem and memBytes .

The memory pointed by mem is cloned by this function.

Parameters
symbolIDIdentifier of the symbol to cache
tagTag of the symbol to identifier
memA pointer to the memory allocated by the caller to store the serialization of the symbol
memBytesNumber of bytes of the memory pointed by mem.

void IRSYMBOL_getSerializationOfSymbolCached ( JITUINT32  symbolID,
JITUINT32 *  tag,
void **  mem,
JITUINT32 *  memBytes 
)

Return the serialization of a symbol.

If the serialization of symbol sym has been previously cached by an invocation of IRSYMBOL_cacheSymbolSerialization , then it stores the information about the serialization of symbol sym.

Parameters
symbolIDIdentifier of the symbol to consider
tagPointer of the memory location where tag of symbolID will be stored into.
memNew memory initialized with the serialization of symbolID will be allocated and its pointer will be stored in this parameter.
memBytesPointer of the memory location where the number of bytes of mem will be stored into.
JITBOOLEAN IRSYMBOL_isSerializationOfSymbolCached ( JITUINT32  symbolID)

Check whether the serialization of a symbol has been cached or not.

Return JITTRUE if the serialization of the symbol with identifier symbolID has been cached by a previous invocation of IRSYMBOL_cacheSymbolSerialization .

Otherwise, JITFALSE is returned.

Parameters
symbolIDIdentifier of the symbol to cache
Returns
JITTRUE or JITFALSE
