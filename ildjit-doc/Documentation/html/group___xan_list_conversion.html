XanHashTable* xanList_toHashTable ( XanList *  l)

Convert a list to a table.

Convert the list l to a hash table.

For each element of the list (i.e., void *), an equivalent element in the hash table is created; the element and its ID is the void * of the list element.

The caller is in charge to free the returned table whenever he or she does not need it anymore.

For example:

1 XanList *l;
2 XanHashTable *t;
3 
4 // Allocate a list
5 l = xanList_new(malloc, free, NULL);
6 xanList_append(l, mydata);
7 
8 // Convert the just-allocated list
9 t = xanList_toHashTable(l);
10 
11 // Use the hash table
12 codeThatUsesTable(t);
13 
14 // Free the memory
15 xanHashTable_destroyTable(t);
16 xanList_destroyList(l);
