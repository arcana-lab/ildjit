Detailed Description

IR methods can be managed by the API described within this module.

The modules Basic blocks , Instruction , Data type , Method , Program and Loop give you the ability to get information about basic blocks, instructions, data types, methods, the input program and loops respectively.

The modules Instruction , Method and Program give you the ability to modify single IR instructions, IR methods and the input program respectively.

The module Code generation gives you the ability to control how to and when to generate IR or machine code. Moreover, it allows you to execute the produced code.

Dependences across instructions, basic blocks and methods can be found in the module Dependences.

In module Metadata you can find information about how to exploit and extend high level information for IR methods.

Functions that are useful to dump information about IR code are grouped in module Code dumping.

Functions to handle IR symbols are grouped in module Symbols
