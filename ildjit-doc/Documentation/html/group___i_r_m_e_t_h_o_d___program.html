JITBOOLEAN IRPROGRAM_doesMethodBelongToALibrary ( ir_method_t *  method)

Check if a method belongs to a library.

Returns JITTRUE if the method belongs to an external library rather than the input program.

Return JITFALSE otherwise.

If ILDJIT cannot demonstrate that method belongs to an external library, then it returns JITFALSE.

XanList* IRPROGRAM_getIRMethods ( void  )

Get the list of all IR methods.

Return the list of methods that have been compiled already by ILDJIT.

Every element of the list is of type (ir_method_t *).

The caller is in charge to free the memory of the list as the following code shows:

1 XanList *methods;
2 
3 // Fetch all methods
4 methods = IRPROGRAM_getIRMethods();
5 
6 // Use all methods
7 ...
8 
9 // Free the memory
10 xanList_destroyList(methods);

XanList* IRPROGRAM_getMethodsWithName ( JITINT8 *  methodName)

Get the set of methods with a given name.

Returns the set of methods that has the same name of methodName .

The name of methods used to compare against methodName is the one returned by

1 IRMETHOD_getMethodName

The caller is in charge to destroy the list returned.

For example:

1 XanList *l;
2 
3 // Fetch the methods with name "main"
4 l = IRPROGRAM_getMethodsWithName("main");
5 
6 // Use the methods
7 item = xanList_first(l);
8 while (item != NULL){
9  ir_method_t *m;
10 
11  // Fetch a method
12  m = item->data;
13  assert(strcmp(IRMETHOD_getMethodName(m), "main") == 0);
14 
15  // Use the method
16  ...
17 
18  // Fetch the next element from the list
19  item = item->next;
20 }
21 
22 // Free the memory used by the list
23 xanList_destroyList(l);
Returns
List of (ir_method_t *)
