Detailed Description

An induction variable.

An induction variable can be computed by the following expression:

a + b * i + pow(c, i)

where i is the basic induction variable from which the current one is derived.

The type of the term 'a': () IROFFSET : 'a' is a variable; () -IROFFSET: 'a' is a negated variable; () IRINT64 : 'a' is a constant value.

The type of the term 'b': () IROFFSET : 'b' is a variable; () -IROFFSET: 'b' is a negated variable; () IRINT64 : 'b' is a constant value.

The type of the term 'c': () IROFFSET : 'b' is a variable; () -IROFFSET: 'b' is a negated variable; () IRINT64 : 'b' is a constant value.

JITINT64 value

If 'a' is a variable, stores its ID, otherwise stores the constant value of 'a'.

If 'b' is a variable, stores its ID, otherwise stores the constant value of 'b'.

If 'c' is a variable, stores its ID, otherwise stores the constant value of 'c'.
