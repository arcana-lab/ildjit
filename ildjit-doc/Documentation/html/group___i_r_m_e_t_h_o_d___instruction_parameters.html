void IRMETHOD_addressOfAccessedMemoryLocation ( ir_instruction_t *  inst,
ir_item_t *  addr 
)

Get the address of the memory location accessed.

Store into addr the address of the memory location accessed by the instruction inst .

If the address is unknown (e.g. it is computed at runtime), then addr is initialized to NOPARAM (i.e., addr->type == NOPARAM).

This is an example of use:

1 ir_item_t mAddr;
2 
3 IRMETHOD_addressOfAccessedMemoryLocation(inst, &mAddr);
4 if (mAddr.type != NOPARAM){
5  printf("The address is %p\n", (void *)(JITNUINT)mAddr.value.v);
6 }
Parameters
instInstruction to consider
addrItem where to store the address
void IRMETHOD_baseAddressAndOffsetOfAccessedMemoryLocation ( ir_instruction_t *  inst,
ir_item_t *  ba,
ir_item_t *  offset 
)

Get the address of the memory location accessed.

Store into ba and offset the base address and the offset from it of the memory location accessed by the instruction inst .

If the base address is unknown (e.g. it is computed at runtime), then ba is initialized to NOPARAM (i.e., ba->type == NOPARAM).

This is an example of use:

1 ir_item_t ba;
2 ir_item_t o;
3 
4 IRMETHOD_baseAddressAndOffsetOfAccessedMemoryLocation(inst, &ba, &o);
5 if (ba.type != NOPARAM){
6  printf("The base address is known at compile time\n");
7 }

instInstruction to consider

offsetItem where to store the offset
ir_item_t* IRMETHOD_getInstructionParameter ( ir_instruction_t *  inst,
JITUINT32  parNum 
)

Get a parameter of an instruction.

Return the parameter parNum of the instruction inst .

Parameter zero represents the return parameter.

Parameters are numbered starting from 1. Hence, the following call returns the first parameter of the instruction:

1 IRMETHOD_getInstructionParameter(inst, 1);
Parameters
instInstruction to consider
parNumNumber of the parameter to consider
Returns
The parameter parNum of the instruction given as input

JITBOOLEAN IRMETHOD_isInstructionParameterAConstant ( ir_instruction_t *  inst,
JITUINT8  parameterNumber 
)

Check if a parameter of an instruction is a constant.

Return JITTRUE if the parameter parameterNumber of the instruction inst is a constant; return JITFALSE otherwise.

The parameter 0 is the return value of the instruction.

Parameters
instIR instruction to consider
parameterNumberNumber of the parameter to consider (it can be 3, 2, 1 or 0)
Returns
JITTRUE or JITFALSE
