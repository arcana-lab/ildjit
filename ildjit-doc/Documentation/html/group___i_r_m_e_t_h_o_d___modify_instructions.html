ir_instruction_t *  inst,
JITBOOLEAN  isVararg 
)

Set or unset the instruction as a call to a vararg method.

If isVararg is JITTRUE, this function set the instruction inst as a call to a vararg method.

Otherwise, it sets inst as not vararg.

Instructions are created as not vararg by default.

Parameters
methodMethod that includes the instruction given as input (i.e., inst)
instInstruction to consider
isVarargJITTRUE or JITFALSE
