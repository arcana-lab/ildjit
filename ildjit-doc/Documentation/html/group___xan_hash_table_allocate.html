Make a new Hash Table.

Allocate and initialize a new Hash table. After the allocation, the hash and equals functions cannot be changed.

Parameters
lengthThe initial size of the hash table. It will grow in size automatically as elements are added.
hasFixedLengthWhether the size of the hash table can grow automatically or not.
allocFunctionThis function is called each time the hash table needs to allocate new memory; if it is NULL, then the hash table will use the malloc function.
reallocFunctionThis function is called each time the hash table needs to reallocate memory; if it is NULL, then the hash table will use the realloc function.
freeFunctionThis function is called each time the hash table needs to free memory; if it is NULL, then the hash table will use the free function.
hashFunctionThis function is called to compute a hash on the key elements; if it is NULL, then the hash table will use a simple default hash function.
equalsFunctionThis function is called to determine whether two keys are equal; if it is NULL, then the hash table will use a simple default comparison function.
Returns
Return the new hash table.
