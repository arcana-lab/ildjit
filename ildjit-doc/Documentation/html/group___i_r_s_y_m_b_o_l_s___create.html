ir_symbol_t* IRSYMBOL_createSymbol ( JITUINT32  tag,
void *  data 
)

Create a new symbol.

Create a new symbol defined as the instance data with the tag tag.

In the system there can be only one pair (tag, data). Therefore if this pair has been already created in the system, calling this function does not create a new one, but it returns the already existing one.

For example, the assertions in the following code are valid:

1 my_symbol_t *mySymbol1;
2 ir_symbol_t *irSymbol1;
3 ir_symbol_t *irSymbol2;
4 JITUINT32 mySymbolTag;
5 
6 // Fetch a free tag
7 mySymbolTag = IRSYMBOL_getFreeSymbolTag();
8 
9 // Allocate my symbol
10 mySymbol1 = allocFunction(sizeof(my_symbol_t));
11 
12 // Set my symbol
13 // ...
14 
15 // Create the correspondent IR symbol
16 irSymbol1 = IRSYMBOL_createSymbol(mySymbolTag, mySymbol1);
17 assert(irSymbol1 != NULL);
18 
19 // Fetch the IR symbol
20 irSymbol2 = IRSYMBOL_createSymbol(mySymbolTag, mySymbol1);
21 assert(irSymbol2 == irSymbol1);
