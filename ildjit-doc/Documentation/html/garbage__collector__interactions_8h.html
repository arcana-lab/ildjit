Detailed Description

ILJIT module to manage the garbage collector plugin

This module control the access to the garbage collector plugin. It known differences beetween memory areas (e.g. it known if a chunk of bytes are an object or an array). Currently it see an allocated memory zone into 3 contiguous blocks: 

  -# an header of size HEADER_FIXED_SIZE
  -# the real object (object fields)
  -# an overhead

It don't show to the garbage collector plugin this memory organization. For futher details see garbage_collector.h.
